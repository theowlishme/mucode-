# 🧩 Problem: Second Largest Element in Array up to Index N

## 📝 Problem Statement
Given a predefined array of integers:

[10, 5, 20, 8, 20, 3]

You are given an integer **N**.

Find the **second largest distinct element** in the array from index **0 to N (inclusive)**.

If the second largest element does not exist, print `-1`.

---

## 📥 Input
- A single integer **N**

---

## 📤 Output
- Print the second largest distinct element from index `0` to `N`
- If it does not exist, print `-1`

---

## 🔒 Constraints
- `0 ≤ N < length of array`

---

## 🧪 Sample Input
4

---

## ✅ Sample Output
10

---

## 🧪 Explanation
Array considered: [10, 5, 20, 8, 20]

Largest = 20  
Second Largest = 10
